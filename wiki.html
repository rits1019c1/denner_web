1. Variables & Types 1. 変数と型

Denner supports both dynamic and static typing. Available types are: Dennerは動的型付けと静的型付けの両方をサポートしています。利用可能な型は次のとおりです： num, str, bool, list, obj. 

// Dynamic typing (Type Inference)
message = "Welcome to Denner"
count = 42
negative_val = -20 // Unary minus supported

// Static typing
age: num = 25
is_admin: bool = true

// Reactive Bindingリアクティブ変数の宣言
score: num observe = 0 // Updates UI automatically

// List Literalsリスト（配列）リテラル
enemies = []
points = [10, 20, 30]

2. Functions 2. 関数

Functions are declared using the function keyword. Parameter and return types are strongly recommended for safety but not strictly required. 関数は function キーワードを使用して宣言します。引数と戻り値の型指定は安全性のために推奨されますが、必須ではありません。 

function greet(name: str): str {
  return "Hello, " + name
}

// Return type is optional (defaults to void)戻り値の型は省略可能です
function spawn() {
  gui.rect(10, 10, 20, 20, "red")
}

// You can also use inline string formatting!文字列への変数の埋め込みも可能です！
function magic(val: num) {
  log.print("Result is: {val * 10}")
}

3. Control Flow & Loops 3. 制御構文とループ

Conditions do not require parentheses. Code blocks must be enclosed in curly braces {}. 条件式に括弧は不要です。コードブロックは波括弧 {} で囲む必要があります。

If / Else

if temperature > 30 {
  log.print("It's hot!")
} else {
  log.print("It's fine")
}

For Loops

// Range Loop範囲のループ
for i in 1..5 {
  log.print(i)
}

// Iterable Loopコレクションのループ
for item in my_list {
  log.print(item)
}

4. Standard Library (Native & JS) 4. 標準ライブラリ

Denner comes with "batteries included" modules that seamlessly translate to Native C++ commands or Browser JS Polyfills dynamically. Dennerには強力な標準ライブラリが用意されており、これらはブラウザ（JavaScript）でも、C++環境でも共通のコマンドで動作します。

Logger ( log.print )

log.print("Outputs to console or terminalコンソール・ターミナルへの文字出力")

Operating System ( os.name, os.env )

Retrieve OS info and access system variables. システムのOS情報を取得したり、環境変数へアクセスします。 

system_name = os.name() // 'browser', 'macos', 'windows'
secret_key = os.env("API_KEY")

Networking ( net.get )

Executes a GET Request. Uses `fetch()` in JS and `libcurl` in C++. Wait-free asynchronous syntax automatically handled! HTTP GET リクエストを実行します。裏側の処理（fetch または curl）はコンパイラが自動で解決するため同期的に書けます！ 

ip = net.get("https://api.ipify.org")

Path ( path.join )

full_path = path.join("folder", "data.json")

User Inputs & Alerts ( cli.input, cli.get_key, gui.alert )

`gui.alert` prompts standard browser modals inside JS and opens a native standalone Notification Window when running inside the Native C++ backend! ユーザー入力を受けたり、通知を出します。C++ネイティブの場合は、なんと本物のOS通知ウィンドウが表示されます！ `cli.get_key` reads raw keyboard input without waiting for "Enter". Perfect for controls! `cli.get_key` は Enterキーを待たずに、押した瞬間にキーを1文字取得します。操作系に最適です。 

user_name = cli.input("What is your name? : ")
key = cli.get_key() // 'Up', 'Down', 'Left', 'Right', 'Space', 'a', etc.
gui.alert("Welcome to Denner, {user_name}!")

Interactive GUI & Games ( gui.setup, gui.loop )

Specialized Canvas-based API for drawing and game loops. Web Playground only. Canvasベースの描画・ゲームループAPIです。Web Playground上でフル機能を利用できます。 

gui.setup(800, 600)
while 1 < 2 {
  gui.clear("white")
  gui.rect(10, 10, 50, 50, "red")

  // Non-blocking key check非ブロッキングなキー入力
  k = gui.get_last_key()
  if k == "Space" {
    log.print("FIRE!")
  }

  gui.loop() // Wait for requestAnimationFrame
}
