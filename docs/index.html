Introduction (1-4)

Denner は、直感的な構文と最新の GUI 物理エンジンを融合させた、高速なスクリプティング言語です。Tauri と React のエコシステムを背景に持ちながら、ユーザーが複雑なインフラを意識することなく、数行のコードで対話的なアプリケーションを作成できるよう設計されています。

Keyword-free

変数宣言に let 等のキーワードは不要です。直接代入するだけで使い始められます。

Proactive GUI

内蔵の物理エンジンにより、衝突や重力、アニメーションを簡単に扱えます。

Reactive State

observe キーワードにより、状態の変化が自動的に UI へ同期されます。
