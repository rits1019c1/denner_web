Setup & CLI (5-14)

Denner は Mac, Linux で動作します。Node.js 環境があれば、コマンド一つで実行可能です。

インストール

npm install -g denner

コマンドリファレンス

CommandEffect
denner run main.denスクリプトを実行します
denner build --exeネイティブ実行バイナリを生成します
denner versionバージョン表示
