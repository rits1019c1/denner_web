Denner is a lightweight programming language that transpiles to high-performance C++ and browser-native JavaScript. Dennerは、高性能なC++とブラウザネイティブのJavaScriptへコンパイルされる、軽量でパワフルなプログラミング言語です。

Get Started 今すぐ始める View Wiki ドキュメントをみる

Dual Backend デュアルバックエンド

Compile to C++14 for native speed or JavaScript for web compatibility. ネイティブ速度のC++14と、Web互換のJavaScript。1つのコードで両方の世界へ。

Zero Dependency ゼロ依存関係

The C++ runtime is header-only and provides safe string interpolation natively. C++ランタイムはヘッダーのみ。文字列補間や数値計算を安全に処理します。

Batteries Included 充実の標準ライブラリ

Built-in modules for OS, Path, Network, and even 2D GUI for Games. OS情報、パス操作、ネットワーク通信、さらにはゲーム用の2D GUIまで標準装備。

Fast Installation 爆速導入

Denner CLI for macOS / Linux / WSL: macOS / Linux / WSL での導入コマンド: 

curl -fsSL https://raw.githubusercontent.com/rits1019c1/denner/refs/heads/main/install.sh | sh

After installation, try denner run hello.den to see the magic. インストール後、denner run hello.den ですぐに実行できます。

Sample Code サンプルコード

Modern Game Loop モダンなゲームループ 

gui.setup(800, 600)
x = 100

for i in 0..100000 {
    gui.clear("black")
    gui.rect(x, 100, 50, 50, "red")
    
    key = cli.get_key()
    if key == "Right" { x = x + 5 }
    
    gui.loop() // Wait for next frame
}
