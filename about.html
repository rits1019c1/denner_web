Denner is a modern programming language with a physics engine, reactive UI, and dual-backend: native C++ and browser JS — both from the same source. Dennerは、物理エンジン・リアクティブUI・C++/JSデュアルバックエンドを備えたモダンなプログラミング言語です。

🎮 Try Live Demo 🎮 ライブデモを見る Get Started 今すぐ始める

Dual Backend デュアルバックエンド

Same source → native C++14 binary or browser JS. One language, two worlds. 同じコードからC++14バイナリとブラウザJSを生成。一つの言語、二つの世界。

Reactive + Physics リアクティブ + 物理演算

observe keyword auto-syncs UI. .enablePhysics() adds gravity in one line. observe でUIを自動同期。.enablePhysics() で一行で重力を追加。

Batteries Included 充実の標準ライブラリ

Built-in os, net, cli, gui modules. Auto HTML export for GUI programs. os・net・cli・guiが標準装備。GUIはワンクリックでHTML書き出し。

🎮 Live Demos 🎮 ライブデモ

These games are built with Denner and run directly in your browser. 以下のゲームはDennerで書かれ、ブラウザでそのまま動作しています。

shooting_game.den

Denner source (shooting game) Dennerソース (シューティング) 

gui.setup(800, 600)

score: num observe = 0  // リアクティブ！

player = gui.rect(375, 520, 50, 30, "#4f46e5")
player.enablePhysics({ gravity: 0 })

while 1 < 2 {
    gui.clear("#0a0a0a")
    gui.text("SCORE: " + score, 20, 70, "#fff")
    gui.loop()
}

Denner source (physics demo) Dennerソース (物理デモ) 

gui.setup(800, 600)

ball = gui.rect(200, 50, 50, 50, "#f472b6")
ball.enablePhysics({ gravity: 9.8 })
ball.on("collision", function() {
    log.print("Bounced!")
})

while 1 < 2 {
    gui.clear("#0a0a0a")
    gui.loop()
}

How to run locally ローカルで実行するには 

# Install Denner
curl -fsSL https://raw.githubusercontent.com/rits1019c1/denner/refs/heads/main/install.sh | sh

# Run the shooting game
denner run https://raw.githubusercontent.com/rits1019c1/denner/refs/heads/main/denner_tests/shooting_game.den

# Build your own game as HTML
denner run mygame.den  # → "y" でHTML出力

Fast Installation 爆速導入

Denner CLI for macOS / Linux / WSL: macOS / Linux / WSL での導入: 

curl -fsSL https://raw.githubusercontent.com/rits1019c1/denner/refs/heads/main/install.sh | sh

After installation, run denner --help to get started. インストール後、denner --help で使い方を確認できます。
